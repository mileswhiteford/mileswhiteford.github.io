Project Goals:¶

For this project we want to explore how weather conditions affect Citi Bike ridership in New York City. We are curious about how temperature, precipitation, wind speed, and snow depth impact the number of daily rides. We will also study the bike dataset independently, identifying common travel hubs and ridership patterns by time of day, gender, and user type.

Project Datasets¶

Dataset 1: New York City Citi Bike Data from December 2019

Source: https://s3.amazonaws.com/tripdata/index.html (2019-citibike-tripdata.zip)

This zip file contains 26 datatsets, each containing per-month citi-bike data, with some months having multiple dataframes. Each dataframe has one row for every citibike ride. This includes 15 columns of information including a description of the ride as well as of the account that took the ride.

The Columns Are:

  • Ride ID
  • Rideable type
  • Started at
  • Ended at
  • Start station name
  • Start station ID
  • End station name
  • End station ID
  • Start latitude
  • Start longitude
  • End latitude
  • End Longitude
  • Member or casual ride

Dataset 2: New York Weather Data Source: NOAA Global Summary of the Day — Central Park Station
URL: https://www.ncei.noaa.gov/data/global-summary-of-the-day/access/2019/72505394728.csv

One row per day containing average temperature, precipitation, wind speed, and snow depth for NYC in 2019. We use this to explore how daily weather conditions drive or suppress Citi Bike ridership.

Proposed Model Questions¶

Model 1: Can daily weather conditions predict daily Citi Bike ridership?

  • Type: k Nearest Neighbors Model
  • Independent Variables (X): Average temperature, precipitation, wind speed, snow depth, day of the week
  • Dependent Variable (Y): Total daily ride count
  • Hypothesis: Temperature will be the strongest positive predictor, while precipitation and snow will be negative predictors

Model 2: Is there a statistically significant difference in trip duration between Subscriber and Customer riders?

  • Type: Two-sample t-test
  • Independent Variable: User type (Subscriber vs. Customer)
  • Dependent Variable: Trip duration in minutes
  • H₀: No significant difference in average trip duration between user types
  • H₁: Customers take significantly longer trips than Subscribers
